{{if eq .Lang "zh-CN"}} 简单化 {{else}} EASY {{end}}

{{if eq .Lang "zh-CN"}} RESTFul支持，MVC架构，可以使用bee快速的开发应用，监控代码修改自动热编译，自动化测试代码，自动化打包部署 {{else}} With RESTFul support, MVC structure, code changes auto complile and reload, auto testing, and auto packing and deploy, you can develop web application very quickly. {{end}}

{{if eq .Lang "zh-CN"}} 智能化 {{else}} SMART {{end}}

{{if eq .Lang "zh-CN"}} 支持智能路由，智能监控，可以监控QPS、内存消耗、CPU使用，goroutine运行状况，让线上的应用都在你的掌握中 {{else}} With smart router support and monitoring you QPS, memory usage, CPU usage, goroutine status by supervisor, you have fully control of your live applicatin. {{end}}

{{if eq .Lang "zh-CN"}} 模块化 {{else}} MODULAR {{end}}

{{if eq .Lang "zh-CN"}} beego内置了强大的模块，支持Session、缓存操作、日志记录、配置解析、性能监控、上下文操作、ORM模块、请求模拟等强大的模块，足以支撑你任何的应用 {{else}} With powerful buildin modules such as session control, caching, logging, configuration parsing, performance supervising, context handling, ORM supporting, and requests simulating, you get the powerful foundation for any type of application. {{end}}

{{if eq .Lang "zh-CN"}} 高性能 {{else}} HIGH PERFORMANCE {{end}}

{{if eq .Lang "zh-CN"}} beego采用了Go原生的http包来处理请求，goroutine的并发效率足以应付大流量的Web应用和API应用，目前已经应用于大量高并发的产品中 {{else}} With native http package to handle the requests and the efficient concurrence of go routine, your Beego application can hanle massive trafic as Beego are doing in many productions. {{end}}
